# Imperial Gallery of Perseia

Premium frontend exploration for a gallery-grade digital experience, built in Next.js and shaped around April 2026 design standards.

## Current status
- Premium editorial homepage foundation is implemented,
- Typography, atmosphere and layout rhythm are intentionally non-template,
- TDD baseline is active with Vitest,
- First critique protocol and round logs are included,
- Gemini CLI critique is currently blocked by auth in this environment.

## Stack
- Next.js 16,
- React 19,
- Tailwind CSS 4,
- shadcn/ui,
- Framer Motion,
- Vitest.

## Local commands
```bash
npm run dev
npm run build
npm run test
npm run lint
```

## Project files
- `PRODUCT_REQUIREMENTS.md` - product and art direction brief,
- `CRITIQUE_PROTOCOL.md` - how Claude Opus and Gemini critique rounds are run,
- `critiques/` - saved critique rounds and blockers,
- `src/config/site.ts` - editable content model,
- `src/app/page.tsx` - homepage implementation.

## Notes
This project is being developed autonomously with frequent commits and GitHub PR flow.
